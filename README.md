# SWARMulator
This project includes a software for simulating swarms in a very simple and uncomplex way.

![SWARMulator](http://logilutions.de/SWARMulator/SWARMulator_3.PNG)

## Shortcuts
* Ctrl + O ==> options
* Ctrl + G ==> generate map
* Ctrl + A ==> spawn ants
* Ctrl + R ==> run simulation
